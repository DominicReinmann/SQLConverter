# SQLConverter

Simple CLI Tool to convert your SQL keywords to Uppercase 

## Installation
```bash
dotnet tool install --global .\nupkg\ SQLConverter
```

## Usage
```bash
sql <filename>
```
|Command|description|
|--|--|
|-f / --f| |

## Update 
```bash
dotnet tool update --global SQLConverter --add-source .\nupkg\
```
